Features


Authentication Scanning

Scan for common misconfigurations of JSON Web Tokens (JWT), OAuth login, session IDs and cookies.

Authorization Analysis

Crawl your site under different user accounts and cross-check the resources available to each user to generate a report allowing you to identify misconfigurations in your Role-Based-Access-Control policy.

Security Best Practices

Generates a list of recommendations to ensure your server is configured according to security best practices.

Form Fuzzing

Optionally fuzz form inputs to submit forms and record resulting API calls. These API calls will later be cross-examined in the Authorization Analysis phase. You can also record manual form submission too.

In-Depth Report

Gain a unique insight into your authorization logic to uncover potential bugs in your code.

How it works


1. Configuration

Enter the details of your site. Record the process for logging in i.e. by typing in username and password fields. Enter user credentials for the users you wish to crawl under - ideally at least two user per role. Add any blacklisted paths you don't want the crawler to touch. Optionally, you can save values to enter for certain forms on your site to ensure they are submitted with valid inputs, otherwise the fuzzer will infer what values to enter.

2. Scanning & Crawling

The scanner logs in to your site and analyzes the method used to authenticate the user as well as checking server configuration details. It then crawls the entire site while logged in as each user in the previously defined list. It fills out forms, using the fuzzer or with pre-recorded form inputs, and follows links. As the crawler runs, API requests are intercepted and logged.

3. Attempt Intrusion

For every API request logged in the previous step, OneScan attempts to perform that request while logged in as every other user. It also attempts each request while not logged in.

4. Reporting

The API requests from step three are parsed and grouped according to the access control pattern and presented along with the authentication and configuration scan results.
